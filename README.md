# MAES | Estudio de Ingeniería 🏗️

**MAES Estudio de Ingeniería** es una web corporativa desarrollada para mostrar los servicios, proyectos y trayectoria de un estudio especializado en diseño, cálculo y verificación estructural.  
El sitio está desplegado en Netlify: [Ver sitio en línea](https://majestic-pasca-4d8e64.netlify.app/).

---

## 🚀 Tecnologías utilizadas
- **HTML5** y **CSS3** para la estructura y estilos.
- **JavaScript** para interactividad y navegación.
- **Netlify** para despliegue y hosting.
- **GitHub** para control de versiones.

---

## 📂 Estructura del proyecto
/maes-web
│── index.html
│── style.css
│── script.js
│── assets/
│   └── imágenes y recursos


---

## ⚡ Características principales
- Diseño responsivo adaptable a móviles y escritorio.
- Secciones claras: Inicio, Servicios, Proyectos, Sobre nosotros, Contacto.
- Galería de proyectos destacados con imágenes y descripciones técnicas.
- Integración con WhatsApp y correo electrónico para contacto directo.
- Presencia institucional con enlaces a redes sociales (Facebook, Instagram, LinkedIn).
